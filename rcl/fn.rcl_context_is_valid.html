pub unsafe extern "C" fn rcl_context_is_valid(
    context: *const rcl_context_t
) -> bool

Return true if the given context is currently valid, otherwise false. **

  • If context is NULL, then false is returned.
  • If context is zero-initialized, then false is returned.
  • If context is uninitialized, then it is undefined behavior.
  • Attribute | Adherence
  • —————— | ———––
  • Allocates Memory | No
  • Thread-Safe | Yes
  • Uses Atomics | Yes
  • Lock-Free | Yes [1]
  • [1] if atomic_is_lock_free() returns true for atomic_uint_least64_t
  • \param[in] context object which should be checked for validity
  • \return true if valid, otherwise false */
